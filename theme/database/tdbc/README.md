# Tcl/Tk SQL Helper

This project provides utility procedures to interact with SQL databases using Tcl DataBase Connectivity (TDBC).

- [Tcl ODBC Driver Documentation](https://core.tcl-lang.org/tdbc/doc/trunk/embedded/www/odbc.html)
